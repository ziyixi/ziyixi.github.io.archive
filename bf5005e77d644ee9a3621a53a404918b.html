ADIOS2 is a package to store large scientific dataset. Specfem uses ADIOS2 as an option to store the mesh files. Meanwhile, ADIOS2 has also provided friendly python API and avoid generating lots of small files. Here I am following the document to compile the package on Stampede2.
wget https://github.com/ornladios/ADIOS2/archive/refs/tags/v2.6.0.tar.gz
tar xvf v2.6.0.tar.gz
cd ADIOS2-2.6.0
mkdir build
cd build
module load phdf5/1.10.4 # not necessary
CC=icc CXX=icpc FC=mpif90 cmake -DADIOS2_USE_Fortran=ON -DADIOS2_USE_Python=OFF -DCMAKE_INSTALL_PREFIX=/scratch/05880/tg851791/libs/adios2 ..
cmake --build . --target install -j 48
 
Note: currently Specfem3D globe doesn't support the newest version, and xspecfem3D will crash for the newer version. V2.6.0 has been tested and is recommended to use.
